# Noel Gafner — Portfolio

Persönliches Developer-Portfolio. Gebaut ohne Framework — reines HTML, CSS & JavaScript.

## Seiten

| Seite            | Beschreibung                        |
|------------------|-------------------------------------|
| `index.html`     | Startseite mit Hero & Übersicht     |
| `about.html`     | Über mich, Skills, Terminal-Block   |
| `projects.html`  | Web-Projekte & GitHub Repositories  |
| `contact.html`   | E-Mail & LinkedIn                   |

## Design

- **Ästhetik:** Terminal Prime — Dark Mode, Electric Green & Cyan Neons
- **Fonts:** Chakra Petch (Headings) · JetBrains Mono (Body)
- **Features:** Dot-Grid-Background, Scanlines, Scroll-Reveal, Glitch-Cursor, Mobile Nav

## Lokal öffnen

Einfach `index.html` im Browser öffnen — kein Build-Schritt nötig.
